# FitTrack and Supplement Laboratory 

This project is a tracking system that enables athletes to monitor their training volumes and supplement (post-workout nutrition) stock levels.

## Features
- **Supplement Tracking:** Stock status, daily recommended dose, and expiration date alerts.
- **Training Log:** Daily set, rep, and weight records.
- **Body Analysis:** Weight and fat percentage change tracking.

## Technologies
- **Database:** SQL Server
- **Backend:** C# / .NET
- **Frontend:** HTML5, CSS3, JavaScript
